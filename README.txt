---SDETOOL---

Calculate the financial viability of your solar panel project with SDE+ subsidies, visualizing expected costs, revenues and earn-back time.

* Investment costs
* Operational costs
* Energy savings
* Estimated subsidies
* Earn-back time
* Expected degeneration

First production project, using Javascript and ChartJS.

This project replaced an expansive Excel spreadsheet, which is included as a downloadable worksheet (SDE_spreadsheet.xlsx)
